# 🧠 LUP · Dashboard Comercial

Dashboard interactivo con los KPIs comerciales de LUP.

**[→ Ver Dashboard](https://agustinmoller-ui.github.io/lup-dashboard/)**

## Qué incluye
- 12 KPIs principales
- Facturación 12 meses por grupo (Richard / Germán / Otros)
- Ticket promedio por pedido y por cliente
- Precio neto/caja vs costo MP/caja y margen directo
- Conversión visita→venta (AV)
- Cajas vendidas vs breakeven
- Cajas/cliente por mes
- % Inactividad de cartera
- Fidelidad distribución
- Cartera histórica vs activos
- EERR Ene–Abr 2026
- Alertas: top clientes con cambio de comportamiento

## Actualización mensual
1. Actualizar `MAESTRA` en el Google Sheet con datos del nuevo mes
2. Correr el script de extracción → genera nuevo `lup_data.json`
3. Subir `lup_data.json` a este repo → dashboard actualizado
